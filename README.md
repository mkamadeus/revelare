`re-vē-lā-re` is a steganography application made for IF4020 Cryptography

## Notable Libraries

- icecream (for debugging)
- flake8 and black (code formatting)

## Setup

```bash
python3 -m venv env
source env/bin/activate # in Windows systems, run `.\env\Scripts\activate`
```

## Testing

```bash
make test
```
